# Agenda de contatos
## About
Esse código é um programa simples de agenda telefônica implementado em Python. Ele permite realizar diversas operações relacionadas a contatos, como inclusão, alteração, exclusão, consulta, listagem e limpeza de contatos. Principais funcionalidades:

Funções de Contato:

- incluirContato: Adiciona um novo contato à agenda se o nome ainda não existir.
- alterarContato: Altera o telefone de um contato existente ou adiciona um novo contato caso o nome não esteja na lista.
- excluirContato: Exclui um contato pelo nome, se o nome estiver na lista.
- consultarContato: Exibe o telefone de um contato específico, se o nome estiver na lista.
- consultarContatos: Lista todos os contatos na agenda, se houver algum.
- limparContatos: Remove todos os contatos da agenda, se houver algum.

Loop Principal:

O programa opera em um loop infinito (while True), onde o usuário pode escolher diferentes opções.
As opções incluem inserir, alterar, excluir, consultar, listar, limpar contatos ou sair do programa.

* Pretendo fazer alterações futuras, como a implementação de um banco de dados para armazenamento.
